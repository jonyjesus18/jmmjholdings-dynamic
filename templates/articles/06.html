Azure Data Facatory Release Pipelines

February 6th, 2022 by Joao

Scenario: You have a Data factory connected to Azure Storage accounts and SQL server. Connections run on managed private endpoints for enhanced security. You want to create CI/CD releases for the Data factory for development, Testing and Production environments. If this rings a bell, this tutorial is for you. I'll be detailing the steps for Release Pipelines in Azure Data factory using the most up to date Microsoft workflows. I'll also cover overwriting data factory connections and parameters so you can flexibly use a standard Data factory and pass it across different environments.
